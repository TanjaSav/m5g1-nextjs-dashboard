#  Next.js App Router Dashboard

A fully functional invoice dashboard built after completing the full 15‑chapter **Next.js App Router Course** https://nextjs.org/learn.  
The project is deployed on Vercel and demonstrates modern Next.js patterns, clean UI, and real-world CRUD functionality.

##  What I Learned (All 15 Chapters)

1. **Getting Started** — project setup and App Router basics  
2. **CSS Styling** — global styles and Tailwind integration  
3. **Optimizing Fonts and Images** — performance-focused asset loading  
4. **Creating Layouts and Pages** — shared layouts and nested routing  
5. **Navigating Between Pages** — accessible client-side navigation  
6. **Setting Up Your Database** — PostgreSQL + Prisma configuration  
7. **Fetching Data** — server-side async data fetching  
8. **Static and Dynamic Rendering** — choosing rendering strategies  
9. **Streaming** — Suspense and streaming server components  
10. **Adding Search and Pagination** — filtering and paginated results  
11. **Mutating Data** — create, update, delete with server actions  
12. **Handling Errors** — custom error boundaries and fallback UI  
13. **Improving Accessibility** — semantic markup and ARIA patterns  
14. **Adding Authentication** — protected routes and login flow  
15. **Adding Metadata** — SEO metadata and dynamic titles
